Quick Start

Configure vcpkg to use this binary cache:

export VCPKG_BINARY_SOURCES="http,http://{{ request.headers.get('host', 'localhost:15151') }}/{'{name}'}/{'{version}'}/{'{sha}'}"

Then run vcpkg install <package> to automatically cache binaries.
